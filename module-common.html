Plan

Daisy's Plan object. Can be retrieved using module:common~SubscriptionProductClient#getPlans.

Type:
  • Object
Properties:
Name Type Description
id string

Daisy ID.
name string

Plan name.
onChainId string

Plan ID in the Ethereum blockchain (internal use of the SDK).
description string

Plan description.
price string

Plan price in tokens.
period number

Number of periods in periodUnit between bill cycles.
periodUnit string

Period unit: DAYS, WEEKS, MONTHS, YEARS.
maxExecutions string

How many times the Plan is executed.
private boolean

If a Plan is private it requires a signature (with a private key) from the authorizer defined in the Subscription Manager.
active string

Is the plan enabled for subscriptions or disabled?.
